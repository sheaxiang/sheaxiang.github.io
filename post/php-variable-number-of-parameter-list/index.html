PHP在用户自定义函数中支持可变数量的参数列表，在PHP5.6及以上的版本中，由语法…语法实现：

function sum(...$numbers)
{
	$a = 0;
	foreach ($numbers as $number) {
		$a += $number;
	}
	return $a;
}
echo sum(1,2,3,4);

输出结果为：

10

function add($a, $b) {
	return $a + $b;
}

echo add(...[1, 2])."\n";

echo add(...$a);
